# Ashu
![screenshot](./docs/screenshot.png)

> H is a social media network, clone of twitter, where users can chat mainly about Human Rights

## Built With

- Ruby v2.7.0
- Ruby on Rails v6.0.2

## Live Demo

[See Demo](https://thawing-wave-61294.herokuapp.com/) 

## Features
* Devise Authentication and Registration 
* Tweet Comment and Like
* User Following
* Profile page
* Cover Image and profile image upload

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 2.7.0
Rails: 6.0.2


### Setup
First, clone the repo to your computer

```
git clone git@github.com:tigamadou/hrt.git
```

Browser to the cloned folder
```
cd hrt
```

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```



### Usage

Start server with:

```
    rails s
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec --format documentation
```
